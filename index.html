JavaScript History

Origin of JavaScript

JavaScript, initially named Mocha, was developed by Brendan Eich while working at Netscape Communications Corporation in 1995. It was first introduced in Netscape Navigator 2.0 in September 1995. Originally conceived as a lightweight scripting language to complement Java, it underwent rapid iterations and was eventually renamed JavaScript. Its primary purpose was to enable dynamic content on web pages.

Evolution of JavaScript

  • ECMAScript standardization
  • Introduction of frameworks like jQuery, Angular, React
  • Modernization with ES6+ features
Version Release Year Notable Features
ES5 2009 Strict mode, JSON support
ES6 (ES2015) 2015 Arrow functions, classes, let/const
ES7+ 2016 onwards Async/await, spread/rest operators
